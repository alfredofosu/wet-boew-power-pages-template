Modèle Power Pages de la Boîte à outils de l'expérience Web (BOEW)

Aperçu

Ce modèle intègre la Boîte à outils de l'expérience Web (BOEW) avec le modèle de données amélioré de Power Pages, fournissant une base rationalisée pour les applications Web du gouvernement canadien.

Informations sur les versions (en date du mois d'août 2025)

Cadre de travail BOEW v4.0.85
Modèle d'application Web GC v16.2.0
Bibliothèque CSS SGDC v5.0.4
Dépendance jQuery v2.2.4
​ ​Bootstrap v4 (BOEW) / v5 (Power Pages)
​ ​Power Pages v9.7.7.11 (Modèle de données amélioré)

Décisions architecturales

Implémentation SGDC statique

Ce modèle utilise une implémentation statique de la Solution de gabarits à déploiement centralisé (SGDC) plutôt qu'un chargement dynamique. Cette approche a été choisie pour :

  • Éliminer les délais de rendu
  • Prévenir les conflits de bibliothèques avec les composants Power Pages prêts à l'emploi
  • Assurer une performance constante

Compatibilité des versions Bootstrap

Le modèle utilise actuellement Bootstrap v4 pour maintenir la compatibilité avec les bibliothèques BOEW. Bootstrap v5 de Power Pages a été désactivé et peut être réactivé une fois que les bibliothèques BOEW seront mises à jour pour prendre en charge la version plus récente.

Avantages

  • Développement accéléré : Se concentrer sur les exigences commerciales plutôt que sur l'intégration du cadre de travail
  • Maintenance facile : Gestion simplifiée des bibliothèques et parcours de mise à niveau
  • Sans conflit : Résolution des problèmes de compatibilité entre les bibliothèques BOEW et Power Pages

Considérations futures

La pile de bibliothèques BOEW peut être mise à niveau au besoin. Surveillez les mises à jour du cadre de travail BOEW pour la compatibilité Bootstrap v5 afin de tirer parti des capacités de style améliorées de Power Pages.
